Most of the following projects are a work in progress. If you find any of them useful you can add them a star, fork it or buy me a beer!

Hexagon

Hexagon is a micro services framework that doesn't follow the flock. It is written in Kotlin and takes care of: communications, serialization, storage, events, configuration, logging and scheduling. You can think of it as the atoms of your platform.

Blacksheep

Sabina's port in Kotlin. Based on Sabina's features, but written from scratch

Sabina

Sinatra inspired framework for Java 8. A micro web framework for micro services.

MrBean

Forget about Java boilerplate code with this IntelliJ IDEA plugin

Storyboard

Behaviour driven development with Java. Write your test almost like text specifications.
